[IN PROGRESS] Webscraping houses in Los Angeles

For this project I wanted to analyze and visualize housing data in Los Angeles. When searching for relevent datasets, I came to the unfortunate realization that there where none. Therefor I decided to create my own dataset.

This project is still under construction!
Currently I have a python program webscrapes the html of this houselisting website. I used this data to create a dataset and plan on using this to visualize the data found. In order to see the source code of the webscraping and geolocation code you can visit this link.
