COVID-19 Data Exploration Project

In this project I use SQL server to explore worldwide COVID-19 data from ourwordlindata.org . The data was imported into Microsoft SQL Server Management Studio. In SSMS I worked on manipulating and managing the data to retrieve interesting output. To see the SQL code you can click here. In order to see the visualitation look at my Tabblau project.


The steps I took:

  1. Downloading the worldwide covid data from ourwordlindata.org
  2. Creating a database in Microsfot SQL Server Managment Studio and importing the 2 tables CovidDeaths and CovidVaccinations.
  3. Cleaning the data by removing null values and converting the dataTypes where needed.
  4. Looking at the tables and trying to find interesting data from it.
  5. Writing SQL queries using the following Skills: Joining tables, Using Common Table Expressions (CTE's), Windows Functions, Aggregate Functions, Creating Views, Converting Data Types.
  6. Checking the results of the queries against other databases and recourses such as the Data Repository by the Center for Systems Science and Engineering (CSSE) at Johns Hopkins University and the data provided from Rijksoverheid.
